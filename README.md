# NewsMorph v2 (Content Intelligence OS)

Repository ini telah diubah fungsinya menjadi rumah untuk **NewsMorph v2**.
File PHP sebelumnya telah dikosongkan karena proyek ini bermigrasi penuh ke arsitektur Python & Supabase.

## Supabase Auto-Sync
Repository ini terhubung langsung dengan proyek Supabase. Setiap perubahan pada folder supabase/migrations/ akan secara otomatis diterapkan ke database *production* Anda.
